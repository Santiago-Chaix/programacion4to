# 📘 Programación - 4.º Año

Bienvenido a este repositorio de **Programación de 4.º Año**.

Aquí se encuentran los ejercicios, prácticas, trabajos y proyectos realizados durante la cursada.

---

## 🎯 Objetivos

- Aprender los fundamentos de la programación utilizando **Python**.
- Desarrollar el pensamiento lógico y la resolución de problemas.
- Aplicar buenas prácticas de programación.
- Crear un videojuego como proyecto final integrando los conocimientos adquiridos.

---

## 📚 Contenidos

Durante la materia se trabajarán los siguientes temas:

- Introducción a la programación
- Instalación y uso de Python
- Variables y tipos de datos
- Entrada y salida de datos (`input` y `print`)
- Operadores aritméticos, relacionales y lógicos
- Estructuras condicionales (`if`, `elif`, `else`)
- Bucles (`for` y `while`)
- Funciones
- Listas
- Tuplas
- Cadenas de texto
- Diccionarios

---

## 🎮 Proyecto Final

Una vez finalizados los contenidos introductorios, desarrollaremos un **videojuego en Python**, aplicando los conceptos aprendidos durante el año.

El proyecto incluirá:

- Organización del código
- Uso de funciones
- Manejo de colecciones de datos
- Lógica del juego
- Interacción con el usuario
